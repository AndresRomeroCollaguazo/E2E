==================================
🚀 Configuración del Proyecto
==================================

📌 **Framework utilizado:** SerenityBDD para pruebas automatizadas.  
📌 **Dependencias de Maven:**  
- `serenity-core` → Manejo de pruebas en Serenity.  
- `serenity-junit` → Integración con JUnit.  
- `serenity-cucumber` → Soporte para Cucumber.  
- `junit` → Ejecución de pruebas unitarias.  
- `slf4j-simple` → Manejo de logs en Serenity.  

----------------------------------
📜 **Instrucciones para ejecutar**
----------------------------------
1️⃣ **Requisitos:**  
   - **Sistema Operativo:** Windows 10+  
   - **Java:** JDK 17  
   - **Plugins en IntelliJ IDEA:** Cucumber, Gherkin, Hocon  
   - **Maven:** 3.9.1 configurado en variables de entorno  

2️⃣ **Comandos de instalación:**  
   - Al abrir el IDE, Maven descargará las dependencias automáticamente.  
   - Ejecutar `Runner.java` en `com/swag/pe/utilities/website/Runner`.  

3️⃣ **Repositorio del proyecto:**  
   - 📌 **GitHub:** [E2E Serenity BDD](https://github.com/AndresRomeroCollaguazo/E2E).
